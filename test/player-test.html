Autoplay unmute (browser autoplay policy)

After autoplay-muted start, on the first PLAYING event call unMute() without any user gesture — does the browser autoplay policy permit it? Feeds issue #4 / docs/youtube-player-api.md.

Watch the log for isMuted()=false (permitted) vs isMuted()=true (blocked by autoplay policy).

Captions (unofficial module — feeds issue #6)

YouTube captions are controlled via the unofficial setOption/getOption captions module. All calls are guarded in try/catch — errors log rather than throw. Build with cc_load_policy:1 first, then probe the module.
